# 错别字与排版审查清单

## 一、文字审查

- [ ] 是否有错别字？
- [ ] 是否有多字、漏字？
- [ ] 是否有重复词？
- [ ] 是否有中英文混乱？
- [ ] 是否有内部表达？
- [ ] 是否有不适合客户观看的内容？

## 二、排版审查

- [ ] 标题位置是否统一？
- [ ] 标题字号是否统一？
- [ ] 小标题和正文层级是否清晰？
- [ ] 总结栏宽高是否统一？
- [ ] 卡片是否对齐？
- [ ] 图标风格是否统一？
- [ ] 插图是否过于抢眼？

## 三、客户视角审查

- [ ] 是否一眼能看懂核心观点？
- [ ] 是否信息过密？
- [ ] 是否有明显 AI 模板感？
- [ ] 是否适合正式汇报展示？
